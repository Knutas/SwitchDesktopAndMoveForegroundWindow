# SwitchDesktopAndMoveForegroundWindow

Registers the hot key <kbd>Ctl</kbd> + <kbd>Win</kbd> + <kbd>Alt</kbd> +
<kbd>Left</kbd> / <kbd>Right</kbd> to move the active window along when
switching desktop on Windows. Only way to exit is to kill the process.

## Compatibility

Only works with Windows 11 22H2 and up.

## Run at startup

Add a shortcut to the exe in the Windows startup folder:
`%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup`
